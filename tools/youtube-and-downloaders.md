# YouTube, Web Video, and Download Helpers

Purpose: retrieve metadata, transcripts, or approved public media for summarization and analysis.

## Tools and credits

### yt-dlp

- Source/credit: yt-dlp project
- Repository: https://github.com/yt-dlp/yt-dlp
- License: Unlicense, per repository metadata
- Purpose: feature-rich command-line audio/video downloader supporting many sites.
- Provenance note: yt-dlp is a fork of youtube-dl based on youtube-dlc, per project README.

### FFmpeg

- Source/credit: FFmpeg project
- Website: https://ffmpeg.org/
- Purpose: media conversion and stream processing often used alongside downloaders.

## Public-safe usage policy

Respect copyright, platform terms, and user approval. Prefer transcripts or metadata when that is sufficient.

Do not commit downloaded videos, private URLs, cookies, browser profiles, account-specific headers, or personal watch/download history.

## Safe smoke test

Use tool version checks and public metadata-only checks. Keep downloaded content outside the starter repo.

## Secret boundary

Cookies, account sessions, OAuth tokens, and private platform credentials are secrets. Do not store them in Starter.
