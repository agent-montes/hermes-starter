# Tool Review Checklist

Before adding a tool note or setup script:

- What problem does the tool solve?
- What license applies?
- What runtime dependencies does it install?
- Does it read or write credentials, cookies, browser profiles, OAuth tokens, or session state?
- Does it send data to external services?
- Where does it store config, cache, logs, and output?
- Can it run on public sample data first?
- What is the rollback/removal path?
