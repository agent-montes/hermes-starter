# Hermes Recovery

Public-safe placeholder for recovery process documentation.

In a live setup, this folder is private and may contain backup manifests, encrypted archives, restore procedures, and recovery notes.

Safe to keep in this public template:

- process descriptions;
- placeholder restore steps;
- lists of what must be backed up;
- lists of what must never be published.

Never commit real backup archives, credential recovery codes, session databases, private manifests, request dumps, channel state, or machine-specific recovery material.
