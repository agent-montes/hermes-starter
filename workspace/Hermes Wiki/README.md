# Hermes Wiki

Public-safe placeholder for the durable knowledge-base folder.

In a live setup, this folder may be an Obsidian vault or another structured knowledge base.

Recommended live rules:

- keep a local schema as the source of truth;
- keep an index for navigation;
- keep an append-only log for knowledge-base actions;
- preserve raw evidence;
- protect human-owned notes;
- cite sources for factual claims;
- do not store secrets, credentials, sessions, channel IDs, or recovery archives here.

This placeholder contains no private Wiki content.
