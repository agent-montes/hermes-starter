# Hermes Operations

Public-safe placeholder for operational setup instructions.

In a live setup, this folder is the command center for current operating facts, runbooks, tool provenance, audit notes, and agent context boundaries.

Recommended live contents:

- root README with folder purpose;
- command-center operating manual;
- tool source registry with public provenance;
- report-only update review process;
- audit scripts and checklists;
- agent context mirrors when needed.

Do not store credentials, recovery archives, sessions, private logs, phone numbers, channel IDs, or gateway routing state here.
