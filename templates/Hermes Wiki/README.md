# Hermes Wiki Template

Copy this folder when creating a private Hermes Wiki.

The Wiki stores durable, cited, non-secret knowledge. It is not a secret store and not a runtime backup.

## Start here

1. Rename `SCHEMA.template.md` to `SCHEMA.md`.
2. Rename `index.template.md` to `index.md`.
3. Rename `log.template.md` to `log.md`.
4. Customize the schema for your vault.
5. Keep secrets, credentials, phone numbers, channel IDs, and recovery archives out of the Wiki.

## Suggested folders

- `inbox/` — unprocessed notes.
- `raw/` — immutable source captures, if used.
- `human/` — human-owned writing, if used.
