# Hermes Secrets

Private-only. Do not publish this folder with real contents.

This template repository tracks only this README and blank category templates. In a live setup, store credentials in a local secret store, password manager, keychain, encrypted file, or Hermes-supported auth flow.

Never commit:

- real environment files
- auth files
- API keys
- OAuth tokens
- passwords
- cookies
- phone numbers
- channel IDs
- recovery codes
- messaging credentials
