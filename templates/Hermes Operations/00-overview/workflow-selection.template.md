# Workflow Selection Template

Choose only three workflows to set up first.

## Selection prompt

```text
From this delegation map, help me choose only three workflows to set up first.

I want:
1. One easy workflow that gives me a useful win today.
2. One recurring workflow that removes something I keep checking manually.
3. One high-leverage workflow that could save serious time or improve an important decision.

For each workflow, give me:
- workflow name;
- why this is the right first version;
- exact input needed from me;
- exact output you will produce;
- whether it should be manual, approval-only, or scheduled;
- the smallest test we can run today;
- what would make it worth turning into a skill or scheduled job later.

Do not pick more than three.
```
