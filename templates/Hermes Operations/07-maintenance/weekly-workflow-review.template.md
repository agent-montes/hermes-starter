# Weekly Workflow Review Template

Use this after a week of using the daily assistant loop.

## Review prompt

```text
Review how I used Hermes this week.

Look at workflows we created, tasks I asked for, repeated decisions, and anything that created friction.

Give me:
1. What worked well.
2. What I kept asking for repeatedly.
3. What should become a skill.
4. What should become a scheduled job.
5. What should stay manual.
6. What safety rules need to be clearer.
7. The one workflow we should improve next week.

Keep it practical. I want fewer better workflows, not a giant automation system.
```

## Review fields

- Week of: `<YYYY-MM-DD>`
- Workflows tested: `<LIST>`
- Useful outputs: `<LIST>`
- Repeated asks: `<LIST>`
- Candidate skills: `<LIST>`
- Candidate scheduled jobs: `<LIST>`
- Manual-only tasks: `<LIST>`
- Safety issues or near misses: `<LIST>`
- Workflow to improve next: `<WORKFLOW>`
- Changes approved: `<YES_NO_AND_SCOPE>`

## Rule

Do not promote a workflow to a live skill or scheduled job without explicit approval.
