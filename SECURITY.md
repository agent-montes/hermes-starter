# Security Policy

This starter is public-safe by design. Do not open issues or pull requests that include secrets, tokens, private logs, session data, channel IDs, phone numbers, or recovery material.

## Reporting a problem

If you find private material in a fork or copy, remove it immediately from the affected repository and rotate any exposed credentials. Do not paste the secret into a public issue.

## What belongs here

- Generic setup instructions.
- Placeholder config examples.
- Public-safe scripts.
- Tool review checklists.

## What does not belong here

See `EXCLUDE.md`.
