# Daily Assistant Example

This is a public-safe example. It uses fake placeholder workflows only.

## Example interview outcome

Hermes asks about recurring work and identifies these generic candidate workflows:

1. Summarize public industry news.
2. Prepare a weekly planning brief.
3. Draft email responses for approval.
4. Organize non-sensitive notes.
5. Monitor a public documentation changelog.

## Example delegation map

### Fully autonomous

- Summarize public industry news from approved public sources.
- Format a non-sensitive planning checklist.

### Draft/research then approve

- Draft an email response for the user to review.
- Prepare a GitHub issue draft before posting.

### Remind or prepare context only

- Prepare meeting context from approved notes.
- Remind the user to make a decision.

### Manual-only

- Send sensitive communications.
- Make purchases.
- Delete files.
- Change production systems.

## Example first three workflows

1. Easy win: prepare a daily public-news summary.
2. Recurring check: monitor a public changelog weekly.
3. High-leverage: prepare a decision brief from approved notes.
